# Tarkov Storyline: Prapor Branch Reference

> **Reference document** for updating the storyline map. This covers the **Hand Over to Prapor** main branch.

---

## START NODE

**Recover Armored Case from Woods crashed Plane.**

---

## MAIN BRANCH: HAND OVER TO PRAPOR

### Initial Steps

| Type | Description |
|------|-------------|
| Decision | Hand over Case to Prapor |
| Action | Find compromising material on Prapor on Lighthouse |
| Action | Gain Lightkeeper Access (Network Provider 1 must be available) |
| Cost | Talk to Lightkeeper & pay him 3 TerraGroup Blue Folders |
| Action | Fire Yellow Flare in front of Ultra Mall (Interchange) |
| Action | Kill 15 Targets without dying (One Raid) |
| Action | Open the Case (find Signal Jammer on Labs) |
| Action | 55H Craft at Workbench |
| State | Case is open |

---

## CHOICE (Merge Point)

- **Option A:** Don't work with Kerman
- **Option B:** Work together with Kerman

---

## PATH A: Don't Work with Kerman → Survivor Ending

| Type | Description |
|------|-------------|
| Action | Head to Shoreline Port Entrance (Intercom) & swipe Keycard |
| Outcome | Keycard didn't work, ask Prapor for Help |
| Cost (Discounted) | Prapor wants **300 Million Roubles** (Discounted from 500M because you gave him the Case initially) |
| Action | Find 4 Terragroup report Folders on Labs |
| Action | 5H Craft at Intel Center |
| Action | Give Prapor the Flash Drive with Data |
| Timed Bonus | If done in 72 Hours → **Achievement: "I am Speed"** |
| Action | Kill 50 any Targets on Streets of Tarkov |
| Action | Kill 4 PMCs (Single Raid and survive) |
| Action | Prapor hands you a Note for the Soldiers at the Terminal |
| **ACHIEVEMENT** | **"Easy Way"** |
| FINAL STEP | Head to Shoreline (22:00 - 04:00), talk to Intercom at Tower, Swipe Keycard, approach Terminal (No Weapon in Hand) |
| **ENDING** | **Survivor Ending** |

### Failure Mechanics (Path A)
- If escape fails, buy a new Note for **5 Million Roubles** at Prapor (Limit 2 per Reset)

---

## PATH B: Work Together with Kerman

| Type | Description |
|------|-------------|
| Action | Search for Masterkeycard and RFID Device on Labs |
| State | Keycard acquired, Device not found - talk to Mechanic |
| Cost | Pay Mechanic **40 Bitcoins** |
| Action | Head with Mechanic's Keys to Elektroniks Apartment (Streets) and get RFID Device |
| Action | Activate RFID Card from Armored Case |
| Action | 50H Craft at Intel Center |
| Action | Head to Shoreline Port Entrance (Intercom) & swipe Keycard |

### Sub-Split Choice

- **Option B1:** Help Kerman find Evidence on Terragroup
- **Option B2:** Do NOT help Kerman find Evidence on Terragroup

---

### Path B1: Help Kerman → Savior Ending

| Type | Description |
|------|-------------|
| Action | Complete every other Story Chapter and hand Evidence in |
| Event | Kerman's trusted Contact reaches out (Fence) |
| Requirement | Get **4.0 Fence Reputation** |
| Action (PvE) | Kill 5 PMCs in a Raid without killing Scavs on Shoreline and Interchange |
| Action (PvP) | Use Co-Op Extract with Scav on Woods and Reserve |
| Action | Raise Reputation with BTR Driver to 0.4 (Complete "The Price of Independence") |
| Action | Build Solar Power in Hideout |
| Action | Final Craft for the Keycard |
| FINAL STEP | Head to Shoreline (22:00 - 04:00), talk to Intercom at Tower, Swipe Keycard, approach Terminal (No Weapon in Hand) |
| **ENDING** | **Savior Ending** |

#### Failure Mechanics (Path B1)
- If escape fails, craft Keycard again (11H Craft time, requires new Blank RFID Card)

---

### Path B2: Do NOT Help Kerman → The Betrayal

**Achievement:** "Enough of your Games!"

#### Choice within B2

- **Path B2-i:** Prapor Path / "Will it Blow"
- **Path B2-ii:** Lightkeeper Path → Debtor Ending

---

### Path B2-i: Prapor Path (Reward Path)

> **Note:** This appears to be a reward path rather than an ending path.

| Type | Description |
|------|-------------|
| Action | Talk to Prapor (If Lightkeeper is not unlocked) |
| Cost | Hand over 40 Repair Kits (100% Durability, non-FIR) |
| Cost | Hand over either Kappa, Theta, or Epsilon Secure Container |
| Cost | Hand over 50 Military Components (Virtex, COFDM, etc.) non-FIR |
| Action | Get Bio Weapon Case (Reserve RB-PKPTS) & Hand to Prapor |
| **ACHIEVEMENT** | **"Will it Blow"** |
| **Reward** | Hand over **1 Million USD** |

---

### Path B2-ii: Lightkeeper Path → Debtor Ending

| Type | Description |
|------|-------------|
| **ACHIEVEMENT** | **"U Turn"** |
| Action | Talk to Lightkeeper |
| Action | Find topographic reconnaissance (5 total on 5 different maps, 1 per Map) |
| Action | 6H Craft at Intel Center |
| Action | Bring a Special Flash Drive with Material to Lightkeeper |
| Action | Kill 30 PMCs on Woods & Hand over 100 PMC Dogtags |
| Action | Find Cultist Amulet in each Marked Room |
| Action | Place all Amulets on Shared Bedroom Marked Key on Lightkeeper Island |
| Reward | Lightkeeper hands you a Keycard for Terminal |
| FINAL STEP | Head to Shoreline (22:00 - 04:00), talk to Intercom at Tower, Swipe Keycard, approach Terminal (No Weapon in Hand) |
| **ENDING** | **Debtor Ending** |

#### Failure Mechanics (Path B2-ii)
- If escape fails, barter a new Keycard from Lightkeeper for **1 Blue Folder**
